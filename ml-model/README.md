# SheCare ML/AI Models

This folder is a placeholder for the SheCare Machine Learning models and analytics services. It will be implemented in a future phase.

The ML-model component will be responsible for:
- Predicting Menstrual Cycle Start Dates and Phase Transitions based on history
- Tracking symptom correlations to provide predictive diagnostics and health alerts
- Natural Language Processing (NLP) models for the AI Consultation assistant
- Personalized wellness recommendations and resource curation based on user metrics
